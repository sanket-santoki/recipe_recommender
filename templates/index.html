MyRecipe

Welcome to MyRecipe

MyRecipe is a cutting-edge recipe recommendation system that leverages advanced machine learning techniques to suggest delicious recipes based on the ingredients you have. Our system is built from scratch using Python, Flask, and state-of-the-art NLP tools.

The model is trained on a dataset containing over 48,000 recipes, each complete with detailed nutritional information, ingredient lists, ratings, and reviews. We employ advanced methods such as TF‑IDF (with dimensionality reduction via SVD) and modern embedding techniques. Our hyperparameter tuning and cross‑validation ensure high performance and stability.

Explore the system via the Predictor tab to get personalized recipe suggestions, review our performance metrics in the Results tab, or learn more about the developer in the About Me section.
